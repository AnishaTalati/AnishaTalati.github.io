  • My Final Project MAPA

MAPA

MAPA is a end-to-end project produced in 8 days by me and a team of 3 others. MAPA is a progressive web app that allows users to join and create groups for the purpose of finding their friends at festivals and events. MAPA has two in built maps, a 2D one made using leaflet, and an AR one made using AFRAME, as well as a chat system that utilises Cloud Firestore to store and fetch messages sent from anywhere. You can try out the app HERE or check out the GitHub repository HERE.
